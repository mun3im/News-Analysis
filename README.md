# [News-Analysis](http://www.huseinhouse.com/trendsentiment/)
news analysis system including early-stopping weights

## How-to

1. clone this repository
```bash
git clone https://github.com/Mongoool/News-Analysis.git && cd News-Analysis
```

2. spawn workers
```bash
# worker count
bash gunicorn.sh 1
```
